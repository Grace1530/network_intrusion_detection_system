#  Network Intrusion Detection System (IDS)

A Python-based Intrusion Detection System that monitors network traffic and detects port scanning attacks in real-time.

---

## Features
- Real-time packet sniffing using Scapy
- TCP port monitoring
- Detection of port scanning behavior
- Alert system with attacker IP tracking
- Logging of suspicious activity

---

## Technologies Used
- Python
- Scapy
- Kali Linux
- Nmap (for testing)

---

## How to Run

```bash
sudo python3 packet_sniffer.py
